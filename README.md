# Web-Browser
A simple browser implemented in Java.

## Features
  - Normal Tabs
  - Incognito Tab
  - Browsing history
  
You can also:
  - Surf websites back and forward.
  - Clear browsing history. 
  
## Setup
Requires `Java-8  JDK` or above.

`controlsfx-8.40.11.jar` is required that is included in lib directory.
